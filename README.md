# teufel-it-react
### react version of teufel-it.de

## script jobs
```bash
npm run / yarn
start ----------------- run web-ui, with hot-reload
test ------------------ runs jest
test:watch ------------ runs jest with watch and coverage
build ----------------- build
deploy ---------------- deploy to github pages
```
